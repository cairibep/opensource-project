# Compilador C em Python

Esta é a documentação do  meu**Compilador de C em Python**.  
Este projeto tem como objetivo implementar um compilador simples para a linguagem C utilizando **Python puro**.

## Objetivos do Projeto
- Entender o funcionamento interno de compiladores.
- Implementar as principais etapas: **tokenização, parsing, análise semântica e geração de código assembly**.
