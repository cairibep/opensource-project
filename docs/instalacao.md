# Instalação

## Pré-requisitos
- **Python 3.10+**
- Pip
- Git

## Passos
```bash
# Clone o repositório
git clone https://github.com/SEU_USUARIO/projeto-open-source
cd projeto-open-source

# Crie um ambiente virtual
python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

# Instale dependências (se houver)
pip install -r requirements.txt
```

## Verificação

Execute o comando abaixo para verificar se o compilador roda:

```bash
python main.py testes.c
```
