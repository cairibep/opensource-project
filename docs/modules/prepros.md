# PrePros.py

Implementa um **pré-processador simplificado** para o compilador.  

## Funções principais
- Tratar diretivas como `#define` ou `#include` (caso implementadas).
- Preparar o código-fonte antes da análise léxica.
- Expandir macros simples.

> Obs: Este não é um pré-processador C completo, mas cobre casos básicos para suportar o compilador.
