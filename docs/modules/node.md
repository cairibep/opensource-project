# Node.py

Define a estrutura de **nós da árvore sintática (AST)**.  
Cada nó representa uma construção da linguagem, como expressões, declarações ou instruções.

## Funções/Classes principais
- Classe `Node` → representa um nó genérico da árvore.
- Suporte a diferentes tipos de nós:
  - Expressões aritméticas
  - Declarações de variáveis
  - Estruturas de controle
