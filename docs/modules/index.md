# Documentação dos Módulos

Esta seção descreve os principais módulos que compõem o **Compilador C em Python**.

## Lista de módulos
- [main.py](main.md) → Ponto de entrada do compilador  
- [Tokenizer.py](tokenizer.md) → Analisador léxico (tokenização)  
- [Parser.py](parser.md) → Analisador sintático (parser/AST)  
- [Node.py](node.md) → Estrutura de nós da AST  
- [Token.py](token.md) → Definição de tokens  
- [Definitions.py](definitions.md) → Constantes e definições auxiliares  
- [PrePros.py](prepros.md) → Pré-processador simples  
- [AsmWritter.py](asmwritter.md) → Escrita de código Assembly
